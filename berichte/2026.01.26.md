## Übung (nmap enumeration)
Befehl:
```
nmap -Pn -sS -sV -sC -oN nmap_default.txt 10.10.92.153
```
#### 1. Bedeutung der Nmap-Optionen

| Option             | Bezeichnung       | Bedeutung                                                       |
| ------------------ | ----------------- | --------------------------------------------------------------- |
| `-Pn`              | No Ping           | Überspringt die Host-Erkennung, Ziel wird als online angenommen |
| `-sS`              | TCP SYN Scan      | Halb-offener „Stealth“-Portscan mit SYN-Paketen                 |
| `-sV`              | Version Detection | Erkennt laufende Dienste und deren Versionen                    |
| `-sC`              | Default Scripts   | Führt die Standard-Nmap-Skripte aus                             |
| `-oN`              | Normal Output     | Speichert das Ergebnis als Textdatei                            |
| `nmap_default.txt` | Ausgabedatei      | Name der gespeicherten Scan-Datei                               |
| `10.10.92.153`     | Ziel-IP           | Zu scannendes System                                            |

---

#### 2. Besonderer Port-Scan

| Scan-Typ     | Option | Beschreibung                                            |
| ------------ | ------ | ------------------------------------------------------- |
| TCP SYN Scan | `-sS`  | Sendet nur SYN-Pakete, ohne vollständigen TCP-Handshake |

---

#### 3. Vorteil des verwendeten Scans

| Vorteil                | Erklärung                                        |
| ---------------------- | ------------------------------------------------ |
| Schnell                | Weniger Pakete als bei vollständigem TCP-Connect |
| Unauffällig            | Verbindung wird nicht vollständig aufgebaut      |
| Zuverlässig            | Erkennt offene und geschlossene Ports sehr genau |
| Standard im Pentesting | Häufig verwendeter Scan bei Sicherheitsanalysen  |

---

#### 4. Anzahl der gescannten Ports

| Einstellung   | Anzahl                        |
| ------------- | ----------------------------- |
| Standard-Scan | 1000 TCP-Ports                |
| Portauswahl   | Häufig genutzte Standardports |
